# Merhaba C Kod

Bu repository, C dilinde yazılmış basit bir "Merhaba Dünya" programını içerir.

## Kullanım
- `merhaba.c` dosyasını herhangi bir C derleyicisi ile derleyin.
- Çalıştırın ve ekrana gelen mesajı görün.

## Geliştirme Ortamı
- **İşletim Sistemi:** Windows 11
- **Derleyici:** Embarcadero Dev‑C++ (IDE üzerinde “Derle ve Çalıştır” ile test edilmiştir)
- **Dil Standardı:** Kod ANSI C (C89/90) ile uyumludur; bu sayede hemen her C derleyicisinde (GCC, Clang, Turbo C, vb.) hiçbir değişiklik yapılmadan çalışır.
