# Merhaba C Kod

Bu repository, C dilinde yazılmış basit bir "Merhaba Dünya" programını içerir.

## Kullanım
- `merhaba.c` dosyasını herhangi bir C derleyicisi ile derleyin.
- Çalıştırın ve ekrana gelen mesajı görün.
